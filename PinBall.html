PinBall

What Is PinBall

PinBall is a versus style pinball game that takes inspiration from air hockey while keeping the pinball feel. The project was created using Unreal Engine 5 and primarily using C++ for all its mechanics aside from UI elements.

What I Acomplished

  • Handed input handling for both player's paddles
  • Created a restart system for when a player scores a point
  • Connected the score UI to the when a player gains a point
  • General bug fixing involving UI

Challenges Faced

This project was meant to be my introduction to learning how to code in unreal using C++ so I decided to only use C++ and avoid blueprints. This definitely had a learning curve to start out but once I got the hang of it things progressed smoothly. The major thing I learned from this project is that unreal is made to incorporate a combination of both C++ and blueprints so limiting one's self will impact the efficiency of development.
