Cowboy Home Defense

What Is Cowboy Home Defense

Cowboy Home Defense is a hoard-based shooter with multiplayer capability made in Unreal Engine 5 for the Epic Mega Jam. The game is designed to handle both single-player and online multiplayer functionality.

What I Accomplished

  • Created the foundation for the revolver including shooting, reloading, and swapping
  • Aided in creating network-based multiplayer
  • Acted as a project manager to ensure the project is within scope
  • Created animation and modified model to improve fedback for the player
  • Created a revive system for multiplayer

Challenges Faced

The main challenge I had to deal with would be learning how multiplayer worked for Unreal. Once I got a feeling for server replication and multicasting the process became a bit smoother. The other issue I encountered was learning how to alter a model in Blender as well as importing a Blender animation into Unreal.
